  • Rangy Engine

Rangy is the UCI engine behind Janggi Master. It's a command line program so requires a UCI-compatible graphical user interface (e.g. Winboard)

  1. Download winboard
  2. Run winboard.exe
  3. Select 'Just view or edit game files'
  4. Click 'Load First Engine'
  5. Load rangy_x.x.x_.exe (x.x.x is the version)
  6. Click OK and play
