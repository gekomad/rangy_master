Janggi Master is a web application (server), users can play through the browser against engine or with each other. Two strong engines are available: Rangy and Stockfish.


Options and Features

  • You can play against other player.
  • Janggi puzzle are included.
  • You can limit engine depth.
  • Rich time controls.
  • You can use backward and forward function.
  • You can use the hint function.
  • Game data is automatically saved, so you can try to reproduce it at any time.
  • You can start the game from a specific position.
  • You can import/export a PGN file
  • You can import/export a FEN string.
  • Supports pieces-score display function.
  • There is a version for Android too.
